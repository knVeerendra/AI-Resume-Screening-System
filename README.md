# AI Resume Screening System

AI-powered system to rank resumes based on job description using NLP and semantic similarity.

#  Features
 Single Resume Analysis  
 Bulk Resume Screening  
 AI-based scoring (BERT + Skills + Experience)  
 Candidate ranking  
 Keyword highlighting  

# Tech Stack

- Python  
- Streamlit  
- Sentence Transformers (BERT)  
- Scikit-learn  
- PyMuPDF  

#Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
